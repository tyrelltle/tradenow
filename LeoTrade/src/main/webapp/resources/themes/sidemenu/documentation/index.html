change colors

  1. Open "index.html" file
  2. Go to line 7, or search for: <link rel="stylesheet" type="text/css" href="assets/css/colors/red.css">
  3. Remplace "red" with the color you want:
  • blue
  • darkblue
  • lightblue
  • green
  • orange
  • red
